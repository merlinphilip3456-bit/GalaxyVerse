# GalaxyVerse – Interactive Universe Explorer

## Project Overview

GalaxyVerse is an interactive front-end web project developed using HTML, CSS, and vanilla JavaScript. The project allows users to explore planets in the solar system through visually engaging UI components such as modals, tabs, and accordions.

This project was created as part of a responsive and interactive UI development task focused on improving user experience through JavaScript-based interactivity and reusable components.

---

## Features

* Responsive modern galaxy-themed UI
* Interactive planet cards
* Modal popup system
* Tab navigation (Overview & Details)
* Accordion component for expandable content
* Smooth hover animations and transitions
* Dynamic DOM manipulation using JavaScript
* Organized and reusable code structure
* Mobile-friendly responsive design
* Accessible semantic HTML structure

---

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript

2. Open the folder in Visual Studio Code.
3. Run `index.html` using Live Server or open it directly in a browser.
