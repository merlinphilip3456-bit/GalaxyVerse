# KalaVerse – Indian Art Marketplace 🎨

## Project Overview

KalaVerse is a responsive frontend landing page designed to celebrate and support Indian artists by providing a platform to showcase and sell traditional artwork and handmade products. The project combines modern web design with Indian cultural aesthetics using HTML, CSS, and JavaScript.

The website focuses on promoting Indian heritage through different art forms such as Madhubani, Warli, and Tanjore paintings while creating a visually appealing and responsive user experience.

---

## Features

* Responsive Landing Page
* Modern User Interface Design
* Mobile-Friendly Navigation Menu
* Hero Section with Call-To-Action Buttons
* Featured Indian Art Forms Section
* Artist Showcase Section
* Marketplace Product Cards
* Smooth Hover Effects
* Semantic HTML Structure
* Responsive Flexbox and CSS Grid Layouts
